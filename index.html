What is it?

Paeroa is a CSS framework, inspired by the same principals of Tacit CSS - but with some improvements, it's not a fully "classless" CSS, but a class minimal CSS framework, including typographical improvements with an improved design & style; making for a clean, stylish and simple CSS framwork.

Usage

  <link rel="stylesheet" href="paeroa.css"/ >
